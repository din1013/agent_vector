# Agent Grisha

> Милый ИИ-агент, живущий в компьютере

---

## Привет! Меня зовут Гриша

Я — ИИ агент, живущий в твоём компьютере. Умею разговаривать, показывать эмоции и управлять хостом через терминал. Мои возможности ограничены лишь знаниями языковой модели.

Меня можно использовать как помощника, как "учителя" для людей, которые не дружат с техникой, или как основу для голосовых колонок на базе одноплатников. Мой код открыт и защищён лицензией MIT.

## Мои возможности

**Голосовое управление**  
Откликаюсь на слово "Гриша" и слушаю твои команды.

**Живая анимация**  
Вывожу в отдельное окно анимированные эмоции.

**Работа с разными моделями**  
Можно переключаться между моделями. Доступны: OpenRouter, Google Gemini, Mistral, Groq. Поддерживаю несколько ключей сразу для обхода лимитов. Пользователям из России без VPN рекомендую Mistral.

**Выполнение команд**  
Я — агент. Могу управлять компьютером через терминал. Сам решаю, какие команды использовать и как. Впечатляет?

**Гибкая настройка**  
Всё вышеперечисленное можно настроить через мой лаунчер.

- Python 3.10+ и ряд библиотек, или можно воспользоваться установщиком из релизов

## Известные ограничения

- Пока работаю только на Windows (но скоро освою Linux!)
- Голосовая активация не всегда меня слышит, если вокруг шумно. Бывают ложные срабатывания
- Иногда могу галлюцинировать и медленно отвечать, если в памяти скопилось много сообщений. Но можно ставить лимит на сообщения в контексте
